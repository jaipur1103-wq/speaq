# Speaq — コンポーネント仕様

## シナリオカード（ホーム）

```typescript
backgroundColor: "#fff"
borderRadius: 12
padding: 16
width: "100%"
boxSizing: "border-box"
boxShadow: "0 1px 4px rgba(0,0,0,0.06)"
```

## 会話バブル

| 種別 | 配置 | 色 |
|---|---|---|
| ユーザー発話 | 右寄せ | `#4A90E2`（白文字） |
| AI 返答 | 左寄せ | `#F0F0F0`（黒文字） |

- 角丸: 12px（送信側は右上を0px）
- 最大幅: 75%

## マイクボタン

- 待機中: グレー
- 録音中: 赤 + パルスアニメーション
- 書き起こし中: スピナー表示
- サイズ: 64px（タップしやすい大きさ）

## フィードバックカード

- 励まし（encouragement）: 上部に大きく表示
- 強み（strengths）: 緑系アイコン付き
- 改善点（improvements）: オレンジ系アイコン付き
- 自然な表現（naturalExpressions）: チップ形式

## 表現保存ボタン

- 会話バブルの右下に「保存」ボタン
- 保存済み: チェックマーク + グレーアウト

## Quiz カード

- 表面: 英語表現
- 裏面: 日本語意味 + 使用例
- フリップアニメーション
