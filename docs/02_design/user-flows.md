# Speaq — ユーザーフロー

## メインフロー

```
起動 → / (ホーム)
 ├─ シナリオ選択（既存）
 ├─ シナリオ作成 → /create
 └─ 練習開始 → /practice/[id]
     ├─ マイク入力（音声認識）
     ├─ AI 返答表示
     ├─ 表現を保存（Notebook へ）
     └─ セッション終了 → フィードバック表示
         └─ スコア保存 → /history

/notebook
 ├─ 保存した表現一覧
 └─ Quiz モード → 定着練習

/history → スコア履歴・グラフ

/guide → 使い方説明
```

## 主要UXポイント

- 設定（難易度・トピック・相手スタイル）はホームで完結
- 会話中は「マイクON → 話す → 送信」の3ステップ
- 気になった表現は会話中にその場で保存できる
- フィードバックはセッション全体を1度にまとめて表示
