# Speaq — UX 原則

## 基本姿勢

「英語を話すことへの不安を下げ、達成感を積み重ねる」体験を設計する。

## 原則

### 1. 会話の流れを止めない
- マイク ON/OFF は1タップ
- AI の返答は即時表示（ストリーミング）
- 表現の保存は会話を中断せずにできる

### 2. 達成感を見える化する
- セッション終了後にフィードバックを表示
- 保存した表現の数が積み上がっていくことを示す
- スコア履歴でグロースを確認できる

### 3. 失敗を恐れさせない
- フィードバックは批判でなく「次に使える表現」として提示
- Encouragement（励まし）を必ず含める
- 発音の正誤より「意図が伝わったか」を重視

### 4. 継続を促す
- Quiz で定着を支援（復習サイクル）
- 履歴でモチベーションを維持
- （将来）ストリーク・リマインダーで習慣化

### 5. シンプルさを保つ
- 設定項目を増やしすぎない
- 画面遷移を最小化
- モバイルファーストのレイアウト
