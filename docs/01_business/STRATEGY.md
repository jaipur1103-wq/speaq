# Speaq — プロダクト戦略

最終更新: 2026-03-24

## ミッション

会話するたびに、使える表現が増える。

## コアループ（差別化の核心）

```
会話する → 知らなかった表現を発見する → 記録する → Quiz で繰り返す → 実際に使えるようになる
```

## ターゲット

詳細は `target-persona.md` を参照。

メイン: TOEIC 600〜800点帯の30代ビジネスパーソン。表現の引き出しが少ないと自覚している。

## 基本戦略

大手（ELSA・Duolingo・Cambly）との正面衝突は避ける。
「スピーキング × 表現習得ループ」というニッチに集中する。

## やること・やらないこと

**やること:**
- コアループの精度向上
- 継続動機の設計
- チャンク（キーパターン）学習の効率化

**やらないこと:**
- 発音評価機能
- B2B 営業
- 大規模マーケティング（コアループ検証前）

詳細: `market-research.md`
