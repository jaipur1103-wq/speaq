# Speaq — 収益モデル

最終更新: 2026-03-30

## 現状

無料提供（MVP・コアループ検証フェーズ）

## APIコスト構造（重要）

アプリの主なコストはGroq API（AI）の利用料金。

### 無料枠の上限

| API | 制限 | 実質の上限 |
|---|---|---|
| llama-3.3-70b（テキスト） | 500,000トークン/日 | 約35セッション/日 |
| Whisper（音声） | 7,200秒/時間 | 約72セッション/時間 |

**ボトルネック：テキストAIのトークン上限（35セッション/日）**

### 1セッションのコスト試算（有料プラン移行後）

| 用途 | トークン | 単価（$0.59/100万） |
|---|---|---|
| 1セッション（5ターン）| ~14,200 | 約$0.008（約1.2円） |
| 例文生成（初回） | ~1,200 | 約$0.0007（約0.1円） |
| 話してみる1回 | ~1,100 | 約$0.0006（約0.1円） |

→ **1ユーザーが1日3セッション使っても約3〜4円/日**。月額換算で約100円/ユーザー。

### スケールアップの目安

| ユーザー数 | 状況 | 対応 |
|---|---|---|
| 〜3人 | 無料枠内 | 対応不要 |
| 5〜10人 | 上限に当たり始める | Groq有料プランへ移行（$0.59/100万トークン） |
| 50人 | 月額API費用 約$2,000相当 | 収益化が必要なライン |

---

## 収益化の方向性（検討中）

- フリーミアム（基本無料・月額 ¥500〜1,000）
- シナリオ数・セッション数を無料枠に設定
- プレミアム: 無制限 + 進捗レポート + カスタムシナリオ

## 収益化のタイミング

コアループ継続率が安定してから（DAU・7日継続率を先に計測）。
ユーザーが5〜10人を超えたタイミングでGroq有料プランへの移行も検討。
