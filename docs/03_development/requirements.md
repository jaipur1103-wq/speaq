# Speaq 要件定義書

最終更新: 2026-03-24

---

## 1. プロダクト概要

| 項目 | 内容 |
|---|---|
| アプリ名 | Speaq（スピーク） |
| ターゲット | TOEIC 600〜800点・30代ビジネスパーソン |
| 目的 | ビジネス英語の会話力を実践的に鍛えるスピーキング練習アプリ |
| 言語 | 日本語 / 英語 切り替え対応 |

---

## 2. コアループ

```
シナリオ設定 → 音声入力で会話 → AIフィードバック → 表現保存（Notebook）→ Quiz で定着
```

---

## 3. 機能一覧

### 3-1. シナリオ設定

- 業界・難易度・ペルソナスタイルを選択
- AIがビジネス英語シナリオを自動生成（英語 + 日本語タイトル・概要）
- シナリオ一覧からの再選択

---

### 3-2. 会話（スピーキングセッション）

| 機能 | 内容 |
|---|---|
| 音声入力 | Groq Whisper API で録音 → 書き起こし |
| 相手役AI | シナリオに沿ってAIが英語で返答（日本語訳付き） |
| テキスト入力 | 音声の代替としてキーボード入力も可 |
| 句読点補正 | 音声入力後に自動で句読点を補正 |
| 言語表示切替 | チャットバブルの日本語訳を表示/非表示 |

---

### 3-3. フィードバック

セッション全体（複数ターン）をまとめて評価。1ターンごとの評価は行わない。

| フィールド | 内容 |
|---|---|
| encouragement | セッション全体への印象（1文） |
| strengths | 優れている点（1〜3件、実際のセリフを引用） |
| improvements | 改善点（0〜2件、改善案 + 模範回答） |
| naturalExpressions | 学習価値のある表現（2〜4件） |
| wordCount | 総単語数 |

---

### 3-4. Notebook（表現保存）

- フィードバックから学習価値のある表現を保存
- chunk・chunkDetail・example を一覧表示
- 保存済み表現の削除

---

### 3-5. Quiz（定着）

- Notebook に保存した表現から出題
- 正解・不正解の記録

---

### 3-6. スコア履歴

- 過去のセッション記録（日付・シナリオ・ターン数・フィードバック内容）
- localStorage に保存

---

## 4. 技術スタック

| レイヤー | 技術 |
|---|---|
| フロントエンド | Next.js + TypeScript（App Router） |
| スタイリング | インラインCSS |
| データ保存 | localStorage（認証なし） |
| AI（会話・フィードバック） | Groq（llama-3.3-70b-versatile） |
| AI（音声認識） | Groq Whisper（whisper-large-v3-turbo） |
| AI（翻訳・フォールバック） | Groq（translate APIのみ） |
| ホスティング | Vercel |
| バージョン管理 | GitHub（https://github.com/jaipur1103-wq/speaq） |

---

## 5. 画面構成

```
/ → シナリオ選択・設定
/chat → 会話セッション
/feedback → フィードバック表示
/notebook → 保存済み表現一覧
/quiz → クイズ
/history → スコア履歴
```
