# Speaq — プライバシーポリシー（テンプレート）

最終更新:

## 1. 収集する情報

Speaq は以下の情報を収集します：

- **利用ログ**: セッション履歴、スコア、設定情報
- **音声データ**: 練習セッション中の音声（書き起こし後に破棄）
- **保存した表現**: ユーザーが保存した英語表現・メモ

※ アカウント登録は不要です（localStorage での管理）

## 2. 情報の利用目的

- アプリの機能提供（会話・フィードバック・Quiz）
- サービスの改善

## 3. 音声データの取り扱い

- 音声は Groq Whisper API で書き起こし後、即時破棄されます
- 音声ファイルはサーバーに保存されません

## 4. 使用する外部サービス

- **Groq**（AI・音声認識）: [プライバシーポリシー](https://groq.com/privacy-policy/)
- **Vercel**（ホスティング）: [プライバシーポリシー](https://vercel.com/legal/privacy-policy)

## 5. データの保管

- 学習データは端末のローカルストレージに保存されます
- サーバーへの個人データ送信は最小限です
